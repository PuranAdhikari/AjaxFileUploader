Общая информация:
  
Тип: установка
Языки: мульти, есть русский
Лечение: ручное [keygen-XFORCE + hosts] или автоматическое [PAINTER]
  
Ключи командной строки:

Тихая установка: /S /Q
Выбор места установки: /D=ПУТЬ

Ключ /D=ПУТЬ следует указывать самым последним
Например: файл_установки.exe /S /Q /D=C:\MyProgram
